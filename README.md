# adivinha-numero

Um jogo simples de adivinhação de números desenvolvido em Python.

## Como jogar

1. Certifique-se de ter o Python instalado (versão 3.x).
2. Navegue até a pasta `Jogo da Adivinhacao`.
3. Execute o comando: `python adivinha.py`
4. O jogo irá gerar um número secreto entre 1 e 100.
5. Digite suas tentativas até acertar!

## Requisitos

- Python 3.x

Divirta-se!
